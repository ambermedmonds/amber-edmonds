Web + UX/UI Case Study

Ramen Website

This project focused on translating an early concept into a final responsive website.

When

Oct-Nov 2024

Role

Web Designer

Tools

Figma, HTML/CSS

Objective

Design a story-driven website about instant noodles with clear hierarchy, strong typography, and interactive layouts.

Process
  • Collaborated in a creative director-designer workflow
  • Turned creative direction into a clear page architecture
  • Tested two visual directions for layout and narrative flow
  • Iterated weekly and built selected version in HTML/CSS
  • Refined responsive layouts and breakpoints across devices
Outcome

Delivered a polished, responsive website that balances narrative content with visual style, translating design decisions cleanly into HTML/CSS.
